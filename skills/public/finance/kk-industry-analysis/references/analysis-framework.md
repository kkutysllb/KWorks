## 第一模块：行业整体评估与投资价值定调

### 核心理念

**为什么需要顶层评估？**
- 避免"见树不见林"：在陷入细节前，先用常识判断行业底色
- 快速筛选价值：判断是否值得投入时间深入研究
- 建立分析锚点：后续深入分析围绕核心定调展开

### 五维度雷达评分方法

#### 维度一：天花板与市场空间

**评分标准**：

| 评分 | 标准 | 特征描述 |
|------|------|----------|
| 5星 | 极高天花板 | 万亿级市场，长期增量空间巨大，渗透率<10% |
| 4星 | 高天花板 | 千亿级市场，增量明显，渗透率10-30% |
| 3星 | 中等天花板 | 百亿级市场，存量博弈为主，渗透率30-50% |
| 2星 | 较低天花板 | 市场规模有限，天花板可见，渗透率>50% |
| 1星 | 低天花板 | 小众市场，天花板明确，增长空间有限 |

**判断要点**：
- TAM（总可触达市场）规模
- 增量vs存量：是蛋糕做大，还是抢别人的蛋糕？
- 渗透率曲线：处于S型曲线的哪个位置？
- 国际对标：参考发达国家同行业的发展轨迹

**常见误区**：
- 把"潜在市场规模"等同于"实际可触达市场"
- 忽视替代品的威胁（如：胶卷相机市场曾经很大，但被数码相机替代）
- 线性外推增长率（忽视渗透率饱和后的增速下降）

#### 维度二：商业模式与资本回报

**评分标准**：

| 评分 | 标准 | 特征描述 |
|------|------|----------|
| 5星 | 极佳生意 | 轻资产高壁垒，高ROE(>20%)，强自由现金流，网络效应 |
| 4星 | 好生意 | 轻资产或高壁垒，ROE 15-20%，稳定现金流，品牌溢价 |
| 3星 | 一般生意 | 中等资产强度，ROE 10-15%，现金流波动，需要持续投入 |
| 2星 | 较差生意 | 重资产低壁垒，ROE 5-10%，现金流差，资本开支大 |
| 1星 | 坏生意 | 重资产苦哈哈，ROE <5%，持续失血，价格战激烈 |

**判断要点**：
- 资产属性：轻资产（软件、品牌）vs 重资产（工厂、设备）
- 壁垒来源：技术、品牌、规模、网络效应、牌照
- ROE潜力：毛利率、净利率、资产周转率、杠杆水平
- 现金流特征：是否需要持续大额资本开支？
- 定价权：能否提价而不流失客户？

**商业模式优劣判断框架**：
```
极佳生意的特征：
✓ 高毛利率（>40%）+ 高净利率（>15%）
✓ 轻资产运营（资产周转率高）
✓ 网络效应（用户越多价值越大）
✓ 转换成本高（客户难以离开）
✓ 规模效应（边际成本递减）

差生意的特征：
✗ 低毛利率（<20%）+ 低净利率（<5%）
✗ 重资产运营（资本开支巨大）
✗ 价格战激烈（同质化产品）
✗ 客户转换成本低（随时可换供应商）
✗ 规模不经济（管理复杂度上升）
```

#### 维度三：竞争格局与护城河

**评分标准**：

| 评分 | 标准 | 特征描述 |
|------|------|----------|
| 5星 | 寡头垄断 | 双寡头或寡头格局，头部玩家定价权强，进入壁垒极高 |
| 4星 | 寡头竞争 | 3-5家主导，竞争格局稳定，进入壁垒高 |
| 3星 | 分散竞争 | 多家竞争，格局未定，进入壁垒中等 |
| 2星 | 红海竞争 | 高度分散，价格战激烈，进入壁垒低 |
| 1星 | 完全竞争 | 无差异化，纯价格竞争，零壁垒 |

**判断要点**：
- 市场集中度：CR3、CR5、CR10（前N名市场份额）
- 进入壁垒：资金、技术、牌照、渠道、品牌
- 退出壁垒：资产专用性、沉没成本
- 竞争强度：价格战频率、营销投入占比

**护城河类型识别**：
1. **无形资产**：品牌、专利、牌照（如：高端白酒、医药专利）
2. **转换成本**：客户更换成本高（如：企业软件、医疗器械）
3. **网络效应**：用户越多价值越大（如：社交网络、电商平台）
4. **成本优势**：规模效应、独特资源（如：矿产、规模制造）

#### 维度四：产业生命周期

**评分标准**：

| 评分 | 标准 | 特征描述 |
|------|------|----------|
| 5星 | 成长期爆发 | 渗透率快速提升（10-50%），业绩增速>30%，市场扩容 |
| 4星 | 成长期稳健 | 渗透率稳步提升（30-60%），业绩增速15-30%，格局优化 |
| 3星 | 成熟期 | 渗透率较高（60-80%），业绩增速0-15%，分红价值 |
| 2星 | 衰退前期 | 渗透率饱和（>80%），业绩下滑，行业出清 |
| 1星 | 衰退期 | 市场萎缩，替代品出现，企业退出 |

**生命周期判断框架**：
```
导入期特征：
- 渗透率<10%
- 技术不成熟，产品迭代快
- 企业普遍亏损
- 不确定性极高
→ 投资逻辑：看技术突破、看商业化拐点

成长期特征：
- 渗透率10-60%
- 技术成熟，产品标准化
- 企业业绩爆发
- 投资机会最多
→ 投资逻辑：看渗透率提升、看龙头崛起

成熟期特征：
- 渗透率60-90%
- 行业增速放缓
- 竞争格局稳定
- 龙头分红增加
→ 投资逻辑：看现金流、看分红、看格局优化

衰退期特征：
- 渗透率>90%或下滑
- 替代品出现
- 行业出清
→ 投资逻辑：看剩者为王、看转型机会
```

#### 维度五：宏观与政策顺风

**评分标准**：

| 评分 | 标准 | 特征描述 |
|------|------|----------|
| 5星 | 强顺风 | 国家战略级支持，政策红利持续，契合宏观大势 |
| 4星 | 顺风 | 政策友好，监管环境宽松，符合发展方向 |
| 3星 | 中性 | 政策中性，无明显支持或限制，自主发展 |
| 2星 | 逆风 | 政策收紧，监管加强，发展受限 |
| 1星 | 强逆风 | 政策打压，合规风险高，生存困难 |

**判断要点**：
- 政策导向：是否写入国家战略、五年规划？
- 监管环境：准入门槛、合规要求、牌照管理
- 财政支持：补贴、税收优惠、政府采购
- 国际环境：贸易政策、地缘政治影响

### 最终研判定调方法

**定调原则**：
1. 综合五个维度，而非简单平均
2. 抓住核心矛盾（如：高成长但高风险 vs 低成长但高确定性）
3. 用"常识"检验：这个定调是否符合商业直觉？

**典型定调类型**：

```
高确定性现金牛：
- 特征：高商业模式评分(4-5) + 成熟期(3) + 寡头格局(4-5)
- 案例：高端白酒、免税、水电
- 投资逻辑：赚分红+稳健增长的钱

高赔率成长股：
- 特征：高天花板(4-5) + 成长期(4-5) + 政策顺风(4-5)
- 案例：新能源车、光伏、储能
- 投资逻辑：赚渗透率提升的钱

困境反转机会：
- 特征：成熟期(3) + 竞争格局改善 + 政策边际改善
- 案例：养殖、航运周期股
- 投资逻辑：赚周期反转的钱

高不确定性赛道：
- 特征：导入期(2-3) + 技术不成熟 + 商业模式未验证
- 案例：元宇宙、量子计算
- 投资逻辑：看技术突破，需等待确定性提升
```

## 第二模块：核心驱动与长期确信

### 直接诱因与核心催化识别

**诱因分类**：

1. **政策驱动型**
   - 国家级政策出台（如：双碳目标、国产替代）
   - 地方补贴落地
   - 行业标准确立
   - 监管松绑或收紧

2. **技术突破型**
   - 关键技术突破（如：ChatGPT、固态电池）
   - 成本大幅下降（如：光伏组件降价）
   - 性能跨越式提升

3. **事件催化型**
   - 龙头企业重大动作（如：特斯拉建厂）
   - 标杆项目落地
   - 突发公共事件（如：疫情催生远程办公）

4. **地缘变化型**
   - 供应链重构
   - 贸易政策变化
   - 国际冲突影响

**类比构建技巧**：
- 选择大众熟悉的场景
- 突出核心相似点
- 保留关键差异点

**示例**：
```
行业：AI智能体
诱因：大模型技术突破（GPT-4发布）
类比：AI智能体现在的状态，像极了2007年的智能手机——技术突破已现，生态正在建立，爆款应用蓄势待发
```

### 底层确信度拆解

**三层确信度框架**：

#### Layer 1: 政策周期确信度

**分析方法**：
1. 政策定位：是国家战略、行业规划还是地方政策？
2. 财政支持：真金白银的补贴、税收优惠、采购支持
3. 落地节奏：政策从发布到实施的时间表
4. 持续性：政策是短期刺激还是长期方向？

**判断标准**：
- 高确信：写入国家战略 + 财政支持明确 + 落地时间表清晰
- 中确信：行业规划支持 + 部分补贴 + 落地进度可见
- 低确信：仅政策表态 + 无实质支持 + 落地不明

#### Layer 2: 技术跨越确信度

**分析方法**：
1. 技术成熟度：处于技术成熟度曲线的哪个阶段？
2. 成本下降曲线：成本是否快速下降？下降空间多大？
3. 商业化进度：技术是否已经商用？商用规模如何？
4. 技术路径确定性：是否存在技术路线之争？

**判断标准**：
- 高确信：技术成熟 + 成本大幅下降 + 已规模化商用 + 路径确定
- 中确信：技术可行 + 成本下降中 + 小规模商用 + 路径相对明确
- 低确信：技术不成熟 + 成本高企 + 仅实验室阶段 + 路径未定

#### Layer 3: 供需缺口确信度

**分析方法**：
1. 需求真实性：是真实需求还是政策催生的伪需求？
2. 供给瓶颈：产能扩张速度 vs 需求增长速度
3. 缺口持续性：缺口是短期现象还是结构性问题？
4. 价格信号：价格是否反映供需关系？

**判断标准**：
- 高确信：真实需求爆发 + 产能扩张滞后 + 缺口持续性强
- 中确信：需求增长明确 + 产能逐步释放 + 缺口逐步缓解
- 低确信：需求不确定 + 产能充足 + 无明显缺口

### 区分"业绩支撑"与"情绪催化"

**业绩支撑特征**：
- 有明确的传导路径：技术/政策 → 需求/成本 → 收入/利润
- 可量化验证：订单、销量、收入、利润
- 持续性强：不是一次性事件

**情绪催化特征**：
- 传导路径模糊：仅停留在概念层面
- 难以量化：缺乏具体业绩支撑
- 持续性弱：短期事件驱动

**判断方法**：
```
问自己三个问题：
1. 这个利好能否体现在企业的利润表上？（何时、通过什么路径？）
2. 是否有具体数据可以验证？（订单、销量、收入？）
3. 利好是否具有持续性？（是一次性还是长期影响？）

三个问题都能回答清楚 → 业绩支撑
回答不清楚 → 情绪催化
```

### 不可逆趋势提炼

**判断标准**：
1. **时间维度**：至少5-10年的持续性
2. **空间维度**：全球性或全国性趋势，非局部现象
3. **驱动力量**：由底层结构性因素驱动，非短期扰动

**典型不可逆趋势**：
- 人口结构变化：老龄化、城镇化
- 技术范式迁移：电动化、数字化、智能化、AI化
- 制度性变革：碳中和、国产替代、共同富裕
- 全球秩序重构：供应链重塑、地缘政治

**提炼方法**：
```
步骤1: 列出所有影响行业的长期因素
步骤2: 筛选出"不可逆"的因素（反向思考：什么情况下这个趋势会逆转？）
步骤3: 从不可逆因素中提炼最核心的1个
步骤4: 用一句话表述这个趋势
```

### 逆向思考方法

**思考角度**：

1. **时间错配**：市场预期是否跑在了基本面前面？
   - 市场是否透支了未来3-5年的增长？
   - 当前估值是否已经反映了所有利好？

2. **空间错配**：局部热点是否被误读为全局趋势？
   - 是小众市场还是大众市场？
   - 是区域现象还是全国/全球趋势？

3. **结构错配**：总量增长是否掩盖了结构性分化？
   - 增速是否由少数企业贡献？
   - 行业内部分化程度如何？

4. **周期错配**：短期波动是否被误判为长期趋势？
   - 当前处于周期的哪个位置？
   - 增速是否可持续？

**常见逆向视角**：
- 热门行业的冷思考：估值泡沫、技术瓶颈、政策退坡风险
- 冷门行业的热发现：政策反转、技术突破、供需拐点
- 线性外推的陷阱：增长率不可持续、市场天花板被低估

## 第三模块：产业本质与商业模式

### 核心价值与重估逻辑拆解

**传导路径分析方法**：

```
维度（政策/技术/周期）
    ↓
影响什么？（需求/供给/成本/价格）
    ↓
传导到哪个财务指标？（收入/成本/毛利率/净利率）
    ↓
影响多大？（量化估算）
    ↓
持续性如何？（短期/中期/长期）
```

**传导路径示例**：
```
政策维度：
国产替代政策 → 企业采购国产设备 → 国产设备厂商订单增加 → 收入增长
传导时间：政策发布后6-12个月开始体现
持续性：政策周期内持续（通常3-5年）
量化：政策要求国产化率达到XX%，对应市场规模XX亿元

技术维度：
大模型技术突破 → AI应用成本下降90% → 企业采用AI意愿提升 → SaaS公司ARR增长
传导时间：技术突破后12-24个月开始体现
持续性：技术迭代周期内持续
量化：成本下降XX%，对应市场规模扩大XX倍
```

### 证实与证伪点识别

**核心矛盾点类型**：

1. **技术可用性 vs 商业化成本**
   - 技术可行但成本过高
   - 案例：自动驾驶（技术可行，但商业化成本高）

2. **政策意图 vs 落地进度**
   - 政策表态积极但落地缓慢
   - 案例：氢能（政策支持，但商业化落地慢）

3. **市场需求 vs 支付能力**
   - 需求真实但客户不愿付费
   - 案例：部分企业SaaS（需求存在，但付费意愿低）

4. **产能扩张 vs 需求增长**
   - 产能扩张快于需求增长
   - 案例：光伏（产能过剩风险）

**识别方法**：
```
步骤1: 列出当前市场对该行业的核心假设
步骤2: 对每个假设进行"压力测试"：
  - 这个假设在什么情况下会失效？
  - 是否有数据支持这个假设？
  - 假设的失效概率多大？
步骤3: 识别最容易失效的假设 → 核心矛盾点
```

### 产业界定与边界厘清

**四要素定义法**：

#### 1. 业务范围界定

**方法**：
- 明确做什么、不做什么
- 区分核心业务与边缘业务
- 界定产业链位置

**示例**：
```
行业：AI智能体
业务范围：
- 做：基于大模型的Agent开发平台、垂直场景解决方案
- 不做：大模型基座训练（那是上游的事）
- 边界：与SaaS的区别在于"主动决策能力"
```

#### 2. 核心壁垒识别

**壁垒类型矩阵**：

| 壁垒类型 | 形成原因 | 持续性 | 案例 |
|----------|----------|--------|------|
| 技术壁垒 | 研发投入、专利保护 | 中等（技术可被突破） | 半导体设备 |
| 网络效应 | 用户越多价值越大 | 极强（强者愈强） | 社交平台 |
| 转换成本 | 客户更换成本高 | 强（客户粘性强） | 企业软件 |
| 规模效应 | 边际成本递减 | 强（成本优势） | 制造业 |
| 牌照壁垒 | 政府准入限制 | 强（政策保护） | 金融、医疗 |
| 品牌壁垒 | 长期品牌积累 | 强（心智占领） | 高端白酒 |

#### 3. 痛点解决分析

**B端痛点**：
- 降本：降低人力、运营、生产成本
- 增效：提升效率、产能、质量
- 合规：满足监管要求、规避风险
- 能力补充：获得自身不具备的能力

**C端痛点**：
- 便利性：省时、省力、省心
- 性价比：更便宜、更好用
- 体验升级：更好的体验、更酷的感觉
- 身份认同：身份象征、社交货币

#### 4. 盈利模式拆解

**商业模式画布**：

```
收入来源：
- 产品销售：一次性收费
- 服务收费：按次/按时收费
- 订阅制：按月/年收费
- 分成模式：按交易额分成
- 广告模式：流量变现

成本结构：
- 固定成本：研发、设备、人员
- 变动成本：原材料、营销、渠道
- 规模效应：产量越大，单位成本越低

盈利节点：
- 哪个环节最赚钱？（高毛利率环节）
- 谁掌握定价权？（议价能力强）
- 利润池如何分配？（产业链利润分配）
```

### 多维度产业对比

#### 历史对比

**对比维度**：
- 技术水平：技术成熟度、迭代速度
- 政策环境：政策支持力度、监管严格度
- 市场规模：市场规模变化、增长率变化
- 竞争格局：集中度变化、玩家变化

**方法**：
```
选择对比时点：
- 行业起点（萌芽期）
- 关键转折点（技术突破/政策出台）
- 当前时点

对比分析：
- 哪些因素发生了根本性变化？
- 哪些因素保持稳定？
- 当前处于历史周期的哪个位置？
```

#### 同业对比

**对比维度**：
- 技术路线：不同技术路线的优劣
- 商业模式：不同模式的盈利能力
- 目标市场：不同市场的空间和竞争
- 竞争策略：差异化策略 vs 成本领先策略

**方法**：
```
识别替代方案：
- 完全替代方案：满足相同需求的替代品
- 部分替代方案：满足部分需求的替代品
- 潜在替代方案：未来可能出现的替代品

对比分析：
- 替代方案的威胁程度？
- 转换成本有多高？
- 差异化壁垒在哪里？
```

#### 全球化定位

**对比维度**：
- 发展阶段：国内外发展阶段差异
- 技术水平：技术差距和追赶速度
- 市场规模：国内外市场规模对比
- 政策环境：政策支持力度对比

**定位判断**：
```
领先地位：技术、市场全球领先
  → 投资逻辑：全球扩张、技术输出

追赶地位：技术落后但快速追赶
  → 投资逻辑：国产替代、成本优势

落后地位：技术、市场均落后
  → 投资逻辑：政策保护、市场培育
```

### 双轨制适配原则

**轨道选择标准**：

```
轨道一：实体制造/重资产链条
适用行业：
- 传统制造业（钢铁、化工、建材）
- 新兴制造业（新能源车、光伏、锂电池）

分析重点：
```

### 轨道一：实体制造/重资产链条

#### 上游环节（资源/材料/设备）

**A. 实体与结构**

- 价格弹性：价格波动对供给的影响

**C. 格局与数据**
- 全球格局：龙头企业、市场份额、技术路线
- 国内格局：国产化率、主要玩家、竞争态势
- 关键数据：市场规模、增长率、毛利率、产能缺口

**D. 投资映射**
- 投资主线：资源为王、国产替代、技术突破
- 关键变量：资源价格、产能利用率、国产化进度
- 风险点：价格波动、政策变化、技术替代

#### 中游环节（制造/封装/集成）

**A. 实体与结构**
- 具体形态：制造环节、集成环节、封装环节
- 产品结构：标准化程度、定制化程度
- 技术特点：工艺复杂度、技术迭代速度

**B. 壁垒与议价权**
- 枢纽作用：连接上下游的功能
- 产能释放：产能建设周期、产能利用率
- 良率爬坡：良率提升曲线、量产难度
- 规模效应：规模是否带来成本优势
- 议价能力：对上下游的议价权

**C. 格局与数据**
- 头部玩家：全球龙头、国内龙头、市场份额
- 关键数据：产能规模、产能利用率、良率、毛利率
- 竞争态势：垄断、寡头、分散

**D. 投资映射**
- 投资主线：产能释放、良率拐点、规模效应
- 关键变量：产能利用率、良率、订单可见度
- 风险点：产能过剩、技术迭代、价格战

#### 下游环节（终端/渠道/品牌）

**A. 实体与结构**
- 需求场景：B端/C端应用场景分布
- 场景占比：各场景的市场份额
- 演变趋势：新场景出现、旧场景替代

**B. 壁垒与议价权**

**C. 格局与数据**

- 终端数据：需求增速、渗透率、市场份额
- 消费者画像：目标客户群体、消费能力

**D. 投资映射**

- 风险点：商业化困难、用户流失、竞争加剧

### 咽喉节点识别方法

**定义**：产业链中价值最高、护城河最深、议价权最强的核心节点

**识别标准**：

1. **议价能力强**
   - 能向后压榨供应商
   - 能向前拿捏客户
   - 定价权高

2. **护城河深**
   - 技术壁垒高
   - 资源稀缺
   - 规模效应强
   - 网络效应

3. **价值占比高**
   - 毛利率高
   - 利润池占比大
   - 不可替代性强

**识别方法**：
```
步骤1: 绘制产业链全景图
步骤2: 标注每个环节的毛利率、进入壁垒、议价能力
步骤3: 识别"议价能力最强"的环节 → 咽喉节点
步骤4: 验证：这个环节是否可以向后压榨供应商、向前拿捏客户？
```

### 利润池迁移预判

**利润池迁移驱动因素**：

1. **技术变革**
   - 新技术打破旧格局
   - 案例：智能手机利润从硬件向软件/服务迁移

2. **竞争格局变化**
   - 新玩家进入、旧玩家退出
   - 案例：电商利润从平台向直播带货迁移

3. **政策变化**
   - 政策重塑利益分配
   - 案例：医药集采导致利润从流通环节向创新药迁移

**预判方法**：
```
步骤1: 分析当前利润池分布（各环节利润占比）
步骤2: 识别利润池迁移的驱动因素
步骤3: 判断迁移方向（哪个环节利润占比会提升？）
步骤4: 预判迁移时间（3-5年内会发生什么变化？）
```

## 第五模块：核心风险与长期基本面跟踪

### 颠覆性风险排查

**风险类型**：

#### 1. 技术颠覆风险

**识别方法**：
- 是否存在颠覆性新技术？（性能更好或成本更低一个数量级）
- 技术成熟度如何？（实验室阶段/小规模商用/规模化应用）
- 转换成本多高？（客户是否愿意切换？）

**典型案例**：
```
数码相机颠覆胶卷相机：
- 技术优势：无需胶卷、即时查看、无限复制
- 成本优势：边际成本几乎为零
- 转换成本：低（消费者愿意切换）
→ 柯达破产

新能源汽车颠覆燃油车：
- 技术优势：智能化程度高、驾乘体验好
- 成本优势：用电成本低于用油成本
- 转换成本：中等（需要充电设施）
→ 传统车企转型
```

#### 2. 商业模式颠覆风险

**识别方法**：
- 是否存在更高效的商业模式？（更低的成本、更好的体验）
- 新模式是否可持续？（是否经过验证？）
- 传统模式是否会被替代？（替代程度多大？）

**典型案例**：
```
电商颠覆实体零售：
- 模式优势：无库存压力、更低租金、更广覆盖
- 体验优势：价格透明、选择丰富、送货上门
→ 实体店大量关闭

SaaS颠覆传统软件：
- 模式优势：订阅制降低客户初始投入
- 体验优势：云端部署、自动更新、按需付费
→ 传统软件公司转型
```

#### 3. 跨界打击风险

**识别方法**：
- 行业外巨头是否有进入意愿？（市场是否足够大？）
- 是否具备进入能力？（技术、资金、渠道）
- 进入后是否有竞争优势？（成本、品牌、生态）

**典型案例**：
```
互联网巨头跨界金融：
- 进入意愿：金融市场巨大、利润丰厚
- 进入能力：技术、资金、用户
- 竞争优势：用户体验好、成本低、生态协同
→ 传统银行数字化转型

手机厂商跨界汽车：
- 进入意愿：汽车智能化趋势、市场巨大
- 进入能力：智能化技术、品牌、资金
- 竞争优势：智能化体验、软件定义汽车
→ 传统车企面临挑战
```

### 政策与合规风险

**风险类型**：

1. **监管收紧风险**
   - 行业监管政策变化
   - 合规要求提高
   - 案例：教育培训、互联网平台

2. **国际贸易风险**
   - 出口限制、关税壁垒
   - 供应链断供
   - 案例：半导体设备

3. **环保合规风险**
   - 环保标准提高
   - 碳排放约束
   - 案例：传统制造业

**评估方法**：
```
步骤1: 梳理当前政策环境
步骤2: 识别政策风险点（监管、贸易、环保）
步骤3: 评估风险影响程度（轻微/中等/重大）
步骤4: 制定应对策略（规避/减轻/转移）
```

### 核心验证指标库构建

#### 产业先导指标

**实体制造链条**：
- 核心原材料价格：反映供需关系
- 产能利用率：反映产能紧张程度
- 良率水平：反映技术成熟度
- 库存水平：反映供需平衡
- 价格体系：产品价格、原材料价格

**数字科技链条**：
- 技术迭代节点：版本更新、能力提升
- 开源节点：开源项目发布
- API调用量：反映使用热度
- 开发者数量：反映生态活跃度

#### 市场景气度指标

**通用指标**：
- 订单可见度（Backlog）：在手订单金额、订单交付周期
- 收入增速：营业收入同比增长
- 毛利率变化：反映定价权和成本控制

**B端指标**：
- 客户数量：新增客户、活跃客户
- 客户留存率：续约率、续费率
- ARPU：客单价变化
- NDR（净收入留存率）：反映客户价值增长

**C端指标**：
- DAU/MAU：活跃用户数
- 用户留存率：次日留存、7日留存、月留存
- 付费转化率：免费转付费比例
- 使用时长：用户粘性

**验证指标使用原则**：
- 定期跟踪：周/月/季度定期审视
- 非交易时段：盘后或周末复盘
- 建立基准：设定正常范围和预警阈值
- 趋势判断：关注指标变化趋势而非单点数值

## 常见误区与避坑指南

### 误区1：顶层评估缺失

**表现**：直接跳入细节分析，缺乏对行业整体价值的判断
**风险**：在低价值行业投入大量时间，或错过高价值行业
**避免**：必须先完成第一模块的五维度评估

### 误区2：混淆"潜在市场"与"实际市场"

**表现**：把理论上的市场规模当成实际可触达市场
**风险**：高估市场空间，估值泡沫
**避免**：用渗透率、客户付费意愿、竞争格局修正市场规模

### 误区3：忽视利润池迁移

**表现**：只看当前利润分布，忽视未来变化
**风险**：投资了利润池正在萎缩的环节
**避免**：分析利润池迁移驱动因素，预判未来3-5年变化

### 误区4：双轨制错配

**表现**：用实体制造的分析框架分析数字科技行业
**风险**：关键分析维度遗漏，得出错误结论
**避免**：先判断产业属性，选择正确的分析轨道

### 误区5：验证指标过度依赖盘面数据

**表现**：用股价、成交量等交易指标作为主要验证依据
**风险**：陷入短期波动，忽视长期基本面
**避免**：建立盘后基本面跟踪体系，关注经营数据

### 误区6：逆向思考过度

**表现**：为了逆向而逆向，忽视了真实的产业趋势
**风险**：错过真正的投资机会，或过早做空
**避免**：逆向思考要有数据和逻辑支撑，避免"逆势而动"
