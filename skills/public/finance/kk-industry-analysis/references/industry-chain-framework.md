# 产业链深度解读与价值研判框架 (V3.0)

## 任务目标

- 本技能用于：系统化评估行业投资价值，从顶层定性到产业链拆解，提供结构化的价值研判框架
- 能力包含：多维评估、趋势研判、产业链拆解、咽喉节点识别、风险排查、验证体系构建
- 触发条件：用户要求解读某个行业、评估投资价值、拆解产业链、识别价值高地

## 前置准备

- 确认目标行业/产业链（用户需明确指定分析对象）
- 了解用户关注重点（投资视角/研究视角/业务视角）
- 明确分析深度要求（快速概览 vs 深度研判）
- 判断产业属性（实体制造型 vs 数字科技型）

## 核心分析框架

本技能采用五模块分析框架，强调"顶层评估先行"：

### 第一模块：行业整体评估与投资价值定调

**核心目标**：在陷入细节前，先用"常识"和"商业常识"对行业进行全景定性

**五维度雷达评分**（每个维度1-5星）：
1. **天花板与市场空间**：存量博弈还是增量市场？TAM是否足够大？
2. **商业模式与资本回报**：轻资产高壁垒还是重资产苦哈哈？ROE潜力如何？
3. **竞争格局与护城河**：赢家通吃还是红海肉搏？进入门槛高低？
4. **产业生命周期**：导入期/成长期/成熟期/衰退期？
5. **宏观与政策顺风**：是否契合宏观大势？有无系统性掣肘？

**最终研判定调**：一句话总结长线核心投资价值

**输出要求**：评分有理有据，定调清晰明确

### 第二模块：核心驱动与长期确信

**核心目标**：剥离短期情绪噪音，寻找支撑产业加速的底层逻辑

**关键要素**：
1. **直接诱因与核心催化**：近期事件/政策/技术/地缘变化的直接诱因 + 通俗类比
2. **底层确信度**：政策周期/技术跨越/供需缺口，区分"业绩支撑"与"情绪催化"
3. **不可逆趋势**：提炼1个最核心的长期逻辑基石
4. **逆向思考**：1-2个打破普遍认知的反直觉视角

**输出要求**：类比生动、逻辑分层、逆向视角有洞察力

### 第三模块：产业本质与商业模式

**核心目标**：看透生意的本质，厘清边界与传导路径

**关键要素**：
1. **核心价值与重估逻辑**：政策/技术/周期维度拆解，向利润表传导的真实路径
2. **证实与证伪点**：标注当前逻辑落地的"核心矛盾点"
3. **产业界定与边界**：业务范围、核心壁垒、解决的痛点、盈利模式拆解
4. **多维度产业对比**：历史对比、同业对比、全球化定位

**输出要求**：概念清晰、传导路径明确、边界厘清

### 第四模块：产业链全链路拆解与咽喉节点

**核心目标**：找到产业链中最具价值的高地

**双轨制适配**：
- **轨道一**：实体制造/重资产链条（上游资源→中游制造→下游终端）
- **轨道二**：数字科技/应用软件链条（上游基础设施→中游平台框架→下游终端应用）

**关键要素**：
1. **全景链路图**：清晰勾勒上下游完整链路
2. **产业链"咽喉"**：识别价值最高、护城河最深、议价权最强的核心节点
3. **利润池迁移**：预判未来3-5年利润池流动方向
4. **上下游深度解析**：每个环节必须包含A/B/C/D四大要素
   - A：实体与结构
   - B：壁垒与议价权
   - C：格局与数据
   - D：投资映射

**输出要求**：链路完整、咽喉节点明确、双轨适配

### 第五模块：核心风险与长期基本面跟踪

**核心目标**：建立冷静客观的跟踪体系

**关键要素**：
1. **核心风险排查**：
   - 颠覆性风险：跨界降维打击、新技术/新模式路线
   - 政策与合规风险：全球规则变化、监管收紧
2. **核心验证指标库**：
   - 产业先导指标：原材料价格、产能利用率、技术迭代节点
   - 市场景气度指标：订单可见度、活跃度增速、留存率、库存周转

**输出要求**：风险点明确、验证指标可量化可观察

## 操作步骤

### 标准流程

1. **需求确认与产业属性判断**
   - 确认目标行业和用户关注重点
   - 判断产业属性（实体制造型 vs 数字科技型）
   - 确定产业链拆解轨道

2. **第一模块分析：顶层评估与定调**
   - 五维度雷达评分（每个维度给出核心理由）
   - 最终研判定调（一句话总结投资价值）
   - 验证：是否基于"常识"和"商业常识"？是否避免陷入细节？

3. **第二模块分析：核心驱动解析**
   - 识别直接诱因，构建通俗类比
   - 拆解底层确信度（区分业绩支撑vs情绪催化）
   - 提炼不可逆趋势
   - 提供逆向思考
   - 验证：是否剥离了短期噪音？是否有长期视角？

4. **第三模块分析：产业本质洞察**
   - 核心价值与重估逻辑拆解
   - 标注证实与证伪点
   - 产业界定与边界厘清
   - 多维度对比（历史/同业/全球）
   - 验证：传导路径是否清晰？核心矛盾点是否明确？

5. **第四模块分析：产业链拆解**
   - 绘制全景链路图
   - 识别产业链"咽喉"节点
   - 预判利润池迁移方向
   - 按轨道深度解析上游→中游→下游（每环节含A/B/C/D四要素）
   - 验证：咽喉节点是否明确？数据是否充分？

6. **第五模块分析：风险与验证体系**
   - 排查颠覆性风险与政策风险
   - 构建核心验证指标库（产业先导+市场景气度）
   - 验证：指标是否可量化？是否适合盘后定期跟踪？

7. **报告生成与优化**
   - 按照输出模板生成结构化报告
   - 确保每个模块的数据和逻辑完整性
   - 标注数据来源和可信度
   - 提供可操作的验证指标体系

### 可选分支

- **快速评估模式**：仅输出第一模块（五维评估+定调）和核心结论
- **深度研判模式**：完整执行所有五个模块，提供详细数据支撑
- **产业链聚焦模式**：重点执行第四模块，深度拆解产业链
- **风险验证模式**：重点执行第五模块，构建跟踪体系

## 注意事项

### 分析原则

- **顶层评估先行**：先定调再深入，避免"见树不见林"
- **商业常识优先**：用常识检验逻辑，警惕"伪需求"和"伪市场"
- **长期视角**：剥离短期噪音，关注不可逆趋势
- **逆向思考**：主动寻找反直觉视角，打破共识偏见
- **数据支撑**：关键结论必须有数据支撑，无数据时标注"推测"

### 双轨制适配要点

**实体制造链条**：
- 上游：看资源稀缺性、产能周期、价格弹性
- 中游：看产能释放、良率爬坡、规模效应
- 下游：看需求景气度、渗透率、品牌溢价

**数字科技链条**：
- 上游：看生态定义权、算力资本开支、数据壁垒
- 中游：看开发者生态、编排能力、技术迁移成本
- 下游：看工作流重构、用户留存、变现闭环

### 数据处理规范

- 使用web_search等工具主动收集数据
- 数据来源需标注可信度（官方统计 > 行业报告 > 媒体报道）
- 数据缺失时：再次检索或明确标注"数据暂缺"
- 严禁编造数据

### 风险提示

- 本技能提供分析框架，不构成投资建议
- 投资有风险，决策需谨慎
- 验证指标仅供基本面跟踪参考

## 使用示例

### 示例1：新兴行业分析（数字科技型）

**用户请求**：解读AI智能体产业链

**执行方式**：
1. 第一模块：五维评估AI智能体的投资价值（高成长性、高不确定性）
2. 第二模块：分析大模型技术突破如何催化AI智能体，类比"移动互联网爆发前夜"
3. 第三模块：辨析AI智能体的核心价值（工作流重构），标注商业化落地的核心矛盾
4. 第四模块（轨道二）：拆解基础设施→平台框架→终端应用，识别"模型层"为咽喉节点
5. 第五模块：排查颠覆性风险，建立核心验证指标（用户留存、商业化闭环）

### 示例2：传统行业分析（实体制造型）

**用户请求**：分析锂电池产业链

**执行方式**：
1. 第一模块：五维评估锂电池的投资价值（成长期现金牛特征）
2. 第二模块：分析电动化趋势如何驱动锂电池需求，类比"汽车替代马车"
3. 第三模块：拆解锂电池的盈利模式（规模效应+技术迭代），对比全球格局
4. 第四模块（轨道一）：拆解上游锂矿→中游电池制造→下游整车应用，识别"高端隔膜"为咽喉节点
5. 第五模块：排查新技术路线风险（固态电池），建立产能利用率和库存周转跟踪体系

### 示例3：投资决策支持

**用户请求**：评估半导体设备产业链投资逻辑

**执行方式**：
1. 第一模块：五维评估（高壁垒、国产替代空间大）
2. 第二模块：分析"自主可控"政策周期，区分业绩支撑vs情绪催化
3. 第三模块：拆解半导体设备的商业模式（高研发投入+客户认证壁垒）
4. 第四模块：识别"光刻机"为咽喉节点，预判利润池向国产设备迁移
5. 第五模块：构建验证指标体系（订单可见度、客户认证进度、研发投入占比）
